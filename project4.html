Data Analysis and Machine Learning Project

Textile Dye Supplier Risk Classification

A Streamlit-based web application for analyzing and visualizing risks associated with top textile dye suppliers for 2023-2024.

View on GitHub

Project Overview

This project is designed to help stakeholders in the textile industry monitor and analyze risks related to dye suppliers. It integrates advanced risk classification, sentiment analysis using VADER, and AI-generated recommendations powered by Cohere. The interface offers a modern, dark-themed UI with interactive visualizations built using Plotly.

Key Features

  • Supplier and Risk Filtering for focused analysis.
  • Risk Classification into Geopolitical, Environmental, Financial, Supply Chain, and Market risks.
  • Sentiment Analysis using VADER to determine risk direction.
  • AI-Powered Summaries and Recommendations via Cohere API.
  • Interactive Visualizations for monthly risk trends and sentiment scores.

Technologies Used

  • Streamlit for the web application.
  • Pandas for data manipulation.
  • Plotly for interactive charts.
  • Cohere API for summaries and recommendations.
  • NLTK (VADER) for sentiment analysis.

Data Source

The dashboard uses a JSON dataset (RawDatav4.json) containing articles from 2023-2024 related to top textile dye suppliers, processed to extract risk information.

UI Design

Features a dark theme with custom CSS, responsive layout, interactive elements, and a clean, card-based design for content sections.

Installation

To run this project locally, follow these steps:

  1. Clone the Repository: git clone https://github.com/sharathchandra2004/jk-internships.git then cd jk-internships
  2. Install Dependencies: Ensure Python 3.8+ is installed, then run pip install -r requirements.txt
  3. API Key Setup: Replace the placeholder Cohere API key in finalapp.py with your own key from Cohere.
  4. Run the Application: Use streamlit run finalapp.py to launch the app at http://localhost:8501.
